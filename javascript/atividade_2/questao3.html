3. Validando um formulário. Crie um formulário HTML simples com campos de entrada para nome e e-mail. Use a propriedade onsubmit no formulário para verificar se o nome e o e-mail são preenchidos quando o formulário for enviado. Se algum dos campos estiver vazio, exiba uma mensagem de erro. Se ambos os campos estiverem preenchidos, exiba uma mensagem de sucesso.
